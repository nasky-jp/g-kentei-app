# 教師なし学習

## 概要

**ラベルなしのデータ**からパターンや構造を発見する手法です。「答え」を人が教えるのではなく、データ自体の性質からグループや特徴を抽出します。

---

## 主なタスク

| タスク | 目的 | 代表手法 |
|---|---|---|
| **クラスタリング** | データを類似グループに分ける | k-means、階層的クラスタリング |
| **次元削減** | データの次元数を減らす | PCA、t-SNE、UMAP |
| **異常検知** | 正常から外れたデータを発見 | LOF、Isolation Forest |
| **密度推定** | データの分布を推定 | GMM（混合ガウスモデル）|

---

## クラスタリング

### k-means法

最もシンプルなクラスタリング手法。

1. k個のクラスタ中心（セントロイド）をランダムに初期化
2. 各データ点を最近傍のセントロイドに割り当て
3. 各クラスタの平均でセントロイドを更新
4. 変化がなくなるまで2〜3を繰り返す

**弱点**: kの値を事前に決める必要がある。外れ値に敏感。

### 階層的クラスタリング

データを段階的に統合（凝集型）または分割（分割型）してツリー（デンドログラム）を構築。kを事前に決める必要がない。

---

## 次元削減

### PCA（主成分分析）

データの分散が最大になる方向（主成分）に射影することで次元を圧縮。

- **可視化**（2〜3次元に削減）に使われる
- 情報損失を最小化しながら圧縮

### t-SNE / UMAP

高次元データを2次元に可視化するための非線形手法。クラスタの構造を保ちながら描画できる。PCAよりも視覚的に分かりやすいが、計算コストが高い。

---

## 異常検知

正常データの分布を学習し、そこから外れたデータを「異常」と判定。

| 手法 | 特徴 |
|---|---|
| LOF（局所外れ値因子） | 周辺密度との比較で外れ値スコアを計算 |
| Isolation Forest | ランダム分割で孤立しやすいほど異常 |
| オートエンコーダ | 再構成誤差が大きいものを異常と判定 |

---

## 試験ポイント

> ⚠️ G検定では以下が頻出です

- **k-meansはkを事前指定**する必要がある
- **PCA = 次元削減**の代表手法（教師なし）
- 次元削減の目的：**次元の呪い**の回避・可視化・計算コスト削減
- **異常検知は教師なし学習**で行うことが多い（異常データが少ないため）
- t-SNEは**可視化目的**で使う（距離の絶対値は保持されない）

## 関連用語

| 用語 | 説明 |
|---|---|
| クラスタ | 類似データをまとめたグループ |
| セントロイド | クラスタの中心点 |
| 次元の呪い | 次元が増えるにつれデータが疎になり計算が困難になる現象 |
| デンドログラム | 階層的クラスタリングの結果を示すツリー図 |
| 再構成誤差 | オートエンコーダが入力を再現する際の誤差 |
