# 情報理論

## 概要

**情報理論**はClaudeシャノンが1948年に創設した理論で、情報の定量化・伝達・圧縮を扱う。機械学習では損失関数・モデル評価・特徴選択など幅広い場面で登場し、G検定でも頻出の数学的背景である。

---

## 情報量とエントロピー

### 自己情報量（Self-Information）

事象 x が発生したときの情報量：

$$I(x) = -\log_2 P(x) \quad \text{（ビット単位）}$$

- 確率が低い事象ほど情報量が大きい
- P(x) = 1（確実）なら I(x) = 0（情報なし）

### シャノンエントロピー（Shannon Entropy）

確率変数 X の情報量の期待値（不確かさの指標）：

$$H(X) = -\sum_{x} P(x) \log_2 P(x)$$

- **最大値**：一様分布のとき（すべての事象が等確率）
- **最小値 0**：確実な事象のとき（P(x)=1が1つ）
- 決定木の分割基準（情報利得）にも使われる

| 例 | エントロピー |
|---|---|
| コインが必ず表（P=1） | H = 0 bit |
| 公平なコイン（P=0.5） | H = 1 bit |
| サイコロ（P=1/6均等） | H = log₂6 ≈ 2.58 bit |

> ⚠️ **試験ポイント**
> エントロピーは「不確かさ」「平均情報量」の両方を意味する。一様分布で最大、確定分布で最小（0）。

---

## 結合エントロピーと条件付きエントロピー

### 結合エントロピー

$$H(X, Y) = -\sum_{x,y} P(x,y) \log P(x,y)$$

### 条件付きエントロピー

$$H(Y|X) = -\sum_{x,y} P(x,y) \log P(y|x)$$

- X を知った後の Y の不確かさを表す
- 決定木では：情報利得 = H(Y) - H(Y|X)

---

## 相互情報量（Mutual Information）

2つの確率変数 X・Y が共有する情報量：

$$I(X;Y) = H(X) - H(X|Y) = H(Y) - H(Y|X)$$

$$= \sum_{x,y} P(x,y) \log \frac{P(x,y)}{P(x)P(y)}$$

- X と Y が独立なら I(X;Y) = 0
- 特徴量選択（Feature Selection）でターゲットとの相互情報量が高い特徴を選ぶ
- 常に 0 以上（非負）

---

## KLダイバージェンス（KL Divergence）

2つの確率分布 P・Q の「ずれ」を測る非対称な尺度：

$$D_{KL}(P \| Q) = \sum_x P(x) \log \frac{P(x)}{Q(x)}$$

- 常に 0 以上（D_KL ≥ 0）
- P = Q のとき D_KL = 0
- **非対称**：D_KL(P‖Q) ≠ D_KL(Q‖P)
- VAEの正則化項（潜在分布と正規分布のずれ）
- 強化学習（PPO）の方策更新の制約にも使用

> ⚠️ **試験ポイント**
> KLダイバージェンスは非対称・非負。距離ではなく「ずれ」の尺度。P‖Q と Q‖P は異なる値になる。

---

## 交差エントロピー（Cross-Entropy）

分布 Q で分布 P のデータを符号化したときの平均符号長：

$$H(P, Q) = -\sum_x P(x) \log Q(x)$$

$$= H(P) + D_{KL}(P \| Q)$$

### 損失関数としての交差エントロピー

分類問題では：
- P = 真のラベル（one-hot）
- Q = モデルの予測確率（softmax出力）

$$L = -\sum_k y_k \log \hat{y}_k$$

- 正解クラス以外は 0 なので実質 `L = -log(正解クラスの予測確率)`
- 正解の予測確率が高いほど損失が小さい
- **二値交差エントロピー**（Binary Cross-Entropy）：二値分類で使用

| 予測確率（正解クラス） | 損失 |
|---|---|
| 0.99 | 約 0.01 |
| 0.50 | 約 0.69 |
| 0.01 | 約 4.60 |

> ⚠️ **試験ポイント**
> 交差エントロピー = H(P) + KL(P‖Q)。H(P)は定数なのでKL最小化と交差エントロピー最小化は等価。

---

## 圧縮と符号化

- **シャノンの第一定理（ソースコーディング定理）**：最適な平均符号長 ≥ エントロピー H(X)
- **ハフマン符号**：頻度の高い文字に短い符号を割り当てる最適な接頭符号
- **算術符号化**：ブロック全体を1つの実数として表現
- **情報理論的限界**：エントロピーを下回る圧縮は不可能（無損失の場合）

## 関連用語

| 用語 | 意味 |
|---|---|
| シャノンエントロピー | 確率分布の平均情報量・不確かさの指標 |
| 相互情報量 | 2変数が共有する情報量。独立なら0 |
| KLダイバージェンス | 2分布のずれを測る非対称な指標 |
| 交差エントロピー | 分類タスクの標準的な損失関数 |
| 情報利得 | 特徴量を知ることでエントロピーが減少する量 |
| ハフマン符号 | エントロピーに近い最適な可変長符号化 |
