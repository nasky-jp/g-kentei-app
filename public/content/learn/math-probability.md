# 確率・統計基礎

## 概要

機械学習の理論的基盤となる確率・統計の概念です。「モデルの予測がどの程度確かか」「データがどのような分布をしているか」を理解するために不可欠です。

---

## 基本概念

### 確率とは

**事象が起こる可能性を0〜1の数値で表したもの**。

- 0 = 絶対に起こらない
- 1 = 必ず起こる
- 全事象の確率の総和 = 1

### 条件付き確率

「Bが起きたという条件のもとで、Aが起きる確率」

```
P(A|B) = P(A ∩ B) / P(B)
```

---

## ベイズの定理

**事前確率（Prior）を新しい証拠で更新して事後確率（Posterior）を求める**公式。

```
P(A|B) = P(B|A) × P(A) / P(B)
```

| 用語 | 意味 |
|---|---|
| **事前確率 P(A)** | 証拠を見る前の仮説の確率 |
| **尤度 P(B\|A)** | 仮説が正しいとき証拠が観測される確率 |
| **事後確率 P(A\|B)** | 証拠を見た後の仮説の確率 |

**活用例**: スパムフィルター・医療診断・推薦システム

---

## 確率分布

### 離散分布

| 分布 | 説明 | 例 |
|---|---|---|
| **ベルヌーイ分布** | 0か1の2値確率 | コイン投げ |
| **二項分布** | n回試行での成功回数 | 製品の不良品数 |
| **ポアソン分布** | 単位時間あたりの事象発生回数 | 1時間のアクセス数 |

### 連続分布

| 分布 | 説明 | 特徴 |
|---|---|---|
| **正規分布（ガウス分布）** | 釣り鐘型の分布 | 多くの自然現象・機械学習で最頻出 |
| **一様分布** | 全範囲で同じ確率 | 乱数生成 |
| **指数分布** | 事象間隔の分布 | 故障時間のモデリング |

---

## 統計量

| 統計量 | 説明 | 記号 |
|---|---|---|
| **期待値（平均）** | 確率で重み付けした平均値 | E[X], μ |
| **分散** | 平均からのばらつきの二乗平均 | Var[X], σ² |
| **標準偏差** | 分散の平方根。元の単位でばらつきを表す | σ |
| **共分散** | 2変数の変動の共動き方向 | Cov[X,Y] |
| **相関係数** | 共分散を正規化（-1〜1） | r |

---

## 最尤推定（MLE）

**観測データが得られる確率（尤度）を最大化するパラメータを推定**する方法。

機械学習では、損失関数の最小化と最尤推定は多くの場合同等。

---

## 試験ポイント

> ⚠️ G検定では以下が頻出です

- **ベイズの定理**：事前確率 × 尤度 → 事後確率
- **正規分布の性質**：平均・分散・68-95-99.7則
- **期待値と分散**の計算
- **相関係数は因果関係を示さない**（頻出注意点）
- **最尤推定**：機械学習の損失最小化との関係

## 関連用語

| 用語 | 説明 |
|---|---|
| 標準化（z-score） | 平均0・分散1に変換する変換 |
| 中心極限定理 | n→∞で標本平均は正規分布に近づく |
| p値 | 帰無仮説が正しいときに観測値以上が得られる確率 |
| 有意水準 | 仮説検定の判断基準（通常 0.05）|
| 過分散 | 期待される分散より実データのばらつきが大きい状態 |
