# 生成AI・GAN・VAE・拡散モデル

## 概要

**生成モデル（Generative Model）** は、データの分布を学習し、新しいデータを生成するモデルの総称。画像・音声・テキスト生成の中核技術であり、GAN・VAE・拡散モデル・LLMに発展した。

---

## GAN（Generative Adversarial Network）

2014年にIan Goodfellowが提案。**Generator（生成器）** と **Discriminator（識別器）** を敵対的に学習させる。

### 仕組み

```
ランダムノイズ z
  ↓
Generator G → 偽画像
  ↓
Discriminator D → 本物か偽物か判定
  ↑
本物の画像データ
```

- **Generator**：識別器を騙すように本物らしいデータを生成
- **Discriminator**：本物と偽物を見分ける
- ゼロサムゲームとして交互に学習

### GANの問題点

| 問題 | 内容 | 対策 |
|---|---|---|
| モード崩壊 | Gが同じ出力しか生成しなくなる | WGAN・Conditional GAN |
| 学習不安定 | D/Gのバランスが崩れる | 学習率調整・スペクトル正規化 |
| 評価困難 | 損失と品質の相関が弱い | FID・IS スコア |

### 代表的なGAN

- **DCGAN**：畳み込みを使ったGAN（2015）
- **CycleGAN**：ペアなし画像変換（馬→シマウマ等）
- **StyleGAN**：高品質な顔画像生成。AdaINで細粒度制御（2019）
- **Pix2Pix**：ペアあり画像変換

> ⚠️ **試験ポイント**
> GANのGenerator・Discriminatorの役割と敵対的学習の仕組みは最頻出。モード崩壊の定義も問われる。

## VAE（Variational Autoencoder）

**変分自己符号化器**。通常のAutoencoderの潜在空間に確率分布の制約を加える。

### 通常のAE vs VAE

| | Autoencoder | VAE |
|---|---|---|
| 潜在変数 | 1点のベクトル | 平均μ・分散σの正規分布 |
| 生成 | できない | 分布からサンプリングして生成 |
| 潜在空間 | 非連続で穴がある | 連続・構造化 |

### 損失関数

```
Loss = 再構成誤差 + KLダイバージェンス（正規分布との距離）
```

- 再構成誤差：入力を復元できるか
- KL項：潜在分布が正規分布N(0,1)に近いか（正則化）

### Reparameterization Trick

- z = μ + σ⊙ε（ε〜N(0,1)）で勾配を通す

## 拡散モデル（Diffusion Model）

2020年以降に台頭。**Stable Diffusion・DALL-E・Midjourney** の基盤技術。

### 仕組み

1. **Forward Process（拡散）**：画像に少しずつノイズを加えていく（T ステップ）
2. **Reverse Process（逆拡散）**：ノイズから元の画像を逐次的に復元するモデルを学習

```
元画像 → ノイズを加える（T回） → 完全なガウスノイズ
　　　←　ノイズを除去する（T回）← 学習済みモデルが予測
```

- テキスト条件付きにはCLIPとのクロスアテンションを使用
- GAN比：学習安定・多様性高い。速度は遅い（改善中）

### 代表モデル

| モデル | 開発元 | 特徴 |
|---|---|---|
| Stable Diffusion | Stability AI | オープンソース・潜在空間での拡散 |
| DALL-E 3 | OpenAI | テキスト指示への忠実度が高い |
| Imagen | Google | テキスト品質重視 |

> ⚠️ **試験ポイント**
> 拡散モデルはForward（ノイズ付加）とReverse（ノイズ除去）の2段階。GAN・VAEより学習安定・生成多様性が高い。

## LLM（大規模言語モデル）

- Transformerアーキテクチャをスケールアップ
- **GPT系**：自己回帰型（次のトークンを予測）
- **BERT系**：マスク予測（双方向エンコーダ）
- **T5**：テキスト-to-テキストの統一フレームワーク

### プロンプトエンジニアリング

| 手法 | 概要 |
|---|---|
| Zero-shot | 例なしで指示のみ |
| Few-shot | 数例を示して指示 |
| Chain-of-Thought | 推論ステップを明示 |
| RAG | 外部知識を検索して補完 |

## 関連用語

| 用語 | 意味 |
|---|---|
| GAN | 生成器と識別器の敵対的学習による生成モデル |
| VAE | 確率的潜在空間を持つ自己符号化器 |
| 拡散モデル | ノイズ付加・逐次除去により生成する確率モデル |
| FID | Frechet Inception Distance。画像品質の評価指標 |
| モード崩壊 | GANが多様な出力を生成できなくなる現象 |
| Latent Diffusion | 潜在空間で拡散を行い計算コストを削減する手法 |
