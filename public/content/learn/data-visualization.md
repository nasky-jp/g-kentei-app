# データ可視化・EDA

## 概要

**EDA（Exploratory Data Analysis：探索的データ分析）** は、統計的な仮説を設定する前に、データの特性・分布・外れ値・相関などを視覚的・統計的に把握するプロセス。機械学習において前処理設計・特徴量エンジニアリングの出発点となる重要なステップ。

---

## EDAの手順

1. **データ概要の把握**：行数・列数・データ型・欠損値の確認
2. **単変量分析**：各変数の分布・中心傾向・ばらつきを確認
3. **二変量分析**：変数間の関係・相関を確認
4. **多変量分析**：複数変数の組み合わせパターンを確認
5. **外れ値・異常値の検出**：データ品質の問題を特定
6. **仮説の形成**：モデリングの方針・特徴量候補を絞り込む

---

## 主要な可視化手法

### 分布の把握

| グラフ | 適用場面 | 特徴 |
|---|---|---|
| ヒストグラム | 連続変数の分布 | 階級幅（ビン数）の設定が重要 |
| カーネル密度推定（KDE） | 連続分布の滑らか表示 | ヒストグラムの連続版 |
| 箱ひげ図（Box Plot） | 分布・外れ値の可視化 | 四分位数・外れ値を一目で確認 |
| バイオリンプロット | 分布の形状比較 | 箱ひげ図 + KDEの組み合わせ |
| QQプロット | 正規性の確認 | 直線に乗れば正規分布 |

### 箱ひげ図の読み方

```
  最大値（外れ値除く）
  ─────── ←─ 第3四分位数（Q3, 75パーセンタイル）
  |     |
  |─────| ← 中央値（Q2, 50パーセンタイル）
  |     |
  ─────── ←─ 第1四分位数（Q1, 25パーセンタイル）
  最小値（外れ値除く）
  ● ← 外れ値（Q3+1.5×IQR を超える値）
```

IQR（四分位範囲）= Q3 - Q1

### 関係性の把握

| グラフ | 適用場面 |
|---|---|
| 散布図（Scatter Plot） | 2変数の関係（連続×連続） |
| ヒートマップ | 相関行列・クロス集計の可視化 |
| ペアプロット | 全変数ペアの散布図を一覧表示 |
| バブルチャート | 散布図 + 3つ目の変数をバブルサイズで表現 |
| 折れ線グラフ | 時系列データのトレンド |

> ⚠️ **試験ポイント**
> 箱ひげ図はQ1・Q2（中央値）・Q3と外れ値の定義（1.5×IQR）を覚える。散布図で相関・非線形関係・クラスタを発見できる。

## 相関係数

### ピアソン相関係数 r

$$r = \frac{\sum (x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum (x_i - \bar{x})^2 \sum (y_i - \bar{y})^2}}$$

- -1 ≤ r ≤ 1
- r = 1：完全な正の線形相関
- r = 0：線形相関なし（非線形関係は存在する可能性あり）
- r = -1：完全な負の線形相関
- **前提**：連続変数・線形関係・正規分布

### スピアマン順位相関係数

- 順位に基づく相関。非正規分布・順序データに対応
- 非線形の単調な関係も捉えられる

| r の絶対値 | 相関の強さ |
|---|---|
| 0.7〜1.0 | 強い相関 |
| 0.4〜0.7 | 中程度の相関 |
| 0.2〜0.4 | 弱い相関 |
| 0〜0.2 | ほぼ無相関 |

> ⚠️ **試験ポイント**
> 相関係数≠因果関係。相関があっても疑似相関（第三の変数が原因）の場合がある。

---

## 共分散（Covariance）

$$\text{Cov}(X, Y) = \frac{1}{n}\sum (x_i - \bar{x})(y_i - \bar{y})$$

- 正：同方向に変化する傾向
- 負：逆方向に変化する傾向
- 0：関係なし（ただし非線形関係は存在する可能性あり）
- スケールに依存するため相関係数で正規化して比較

## データ分布の特性把握

### 基本統計量

| 統計量 | 意味 |
|---|---|
| 平均（Mean） | データの重心。外れ値の影響を受ける |
| 中央値（Median） | データの中央。外れ値に頑健 |
| 最頻値（Mode） | 最も多く現れる値 |
| 分散（Variance） | データのばらつきの平方 |
| 標準偏差（SD） | 分散の正の平方根。元データと同単位 |

### 分布の形状

- **歪度（Skewness）**：分布の左右対称性からのずれ（正の歪み = 右に裾が長い）
- **尖度（Kurtosis）**：分布の尖り具合（正規分布比較）

## Pandasプロファイリング

```python
import pandas as pd
from ydata_profiling import ProfileReport

df = pd.read_csv('data.csv')
profile = ProfileReport(df, title="EDA Report")
profile.to_file("eda_report.html")
```

自動的に以下を生成：
- 各変数の分布・統計量
- 欠損値の分布
- 変数間の相関行列
- 重複行の検出

## 関連用語

| 用語 | 意味 |
|---|---|
| EDA | 探索的データ分析。仮説なしにデータを観察・探索 |
| IQR | 四分位範囲（Q3-Q1）。外れ値判定に使用 |
| 歪度 | 分布の非対称性。正規分布なら0 |
| 相関係数 | 2変数の線形関係の強さ（-1〜1） |
| 疑似相関 | 第三の変数が原因の見かけ上の相関 |
| ヒートマップ | 相関行列を色で表現した可視化手法 |
